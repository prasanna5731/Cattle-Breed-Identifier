Build a full-stack web application called "PashuDrishti" (पशु दृष्टि) — an AI-powered image-based breed recognition system for Indian cattle and buffaloes.

---

## TECH STACK
- Frontend: HTML5, CSS3, Vanilla JS (no frameworks)
- Backend: Node.js + Express
- AI: Anthropic Claude API (claude-sonnet-4-20250514) with vision capabilities
- No database needed (stateless)

---

## BACKEND API (server.js with Express)

### POST /api/analyze
- Accepts: multipart/form-data with image file
- Converts image to base64
- Calls Anthropic Claude API with this system prompt:

SYSTEM: "You are an expert veterinarian and animal husbandry specialist with deep knowledge of all Indian cattle and buffalo breeds. When given an image of a bovine, analyze its physical features: body shape, coat color, horn shape/size, hump size (for cattle), ear shape, dewlap, leg structure, and facial features. Identify the most likely Indian breed. Return ONLY a JSON object with this exact structure:
{
  "breedName": "string",
  "hindiName": "string", 
  "species": "Cattle" or "Buffalo",
  "confidence": number (0-100),
  "originState": "string",
  "purpose": "Dairy" or "Draft" or "Dual Purpose",
  "keyFeatures": ["feature1", "feature2", "feature3"],
  "milkYield": "string (e.g. 8-12 liters/day)",
  "bodyColor": "string",
  "hornType": "string",
  "conservationStatus": "Common" or "Vulnerable" or "Endangered",
  "description": "2-3 sentence description",
  "alternateBreeds": ["breed1", "breed2"]
}"

USER: [image + "Identify this Indian cattle or buffalo breed"]

- Parse Claude's JSON response
- Return to frontend

### GET /api/breeds
- Returns full breeds database as JSON

### GET /api/breeds/:id
- Returns single breed details

## FRONTEND JS FEATURES (app.js)

1. Drag & drop image upload with preview
2. Fetch /api/analyze with FormData
3. Animate result section sliding up
4. Animated confidence arc (SVG circle stroke-dashoffset animation)
5. Breed gallery with filter functionality
6. Intersection Observer for stats counter animation
7. Smooth scroll navigation
8. Toast notifications (success/error) — slide in from top-right

## ANIMATIONS (CSS-only where possible)

1. Scan line on upload zone: pseudo-element, top-to-bottom infinite loop, semi-transparent gold line
2. Particle background: 20 tiny divs, random positions, subtle float keyframes
3. Card hover: translateY(-8px) + box-shadow gold glow
4. Result section: slideInUp keyframe
5. Confidence arc: SVG circle animation with JS triggering CSS transition
6. Marquee: CSS animation marquee for breed name ticker
7. Loading spinner: rotating dashed circle in gold
8. Page load: staggered fade-in for hero elements (animation-delay)
